3. 詳細
    1. 「おはようございます」とは  
    起床時、あるいは朝人に会った時の挨拶。
    2. 「こんにちは」とは  
    語源は「今日は、ご機嫌いかがですか？」が省略されたもの
    3. 「こんばんは」とは  
    語源は「今晩は〇〇ですね。」の後半部分が省略されたもの
